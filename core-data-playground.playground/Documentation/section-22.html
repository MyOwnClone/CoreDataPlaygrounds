The stack is now setup and ready to use. Create a new City entity and insert it into the context. This is done by calling insertNewObjectForEntityForName on NSEntityDescription and including the name of the entity and the context that should create it. This will return an NSManagedObject instance for our entity. We can then set attribute values on the entity using key paths. A typical app, with a more complex data model, may create subclasses of NSManagedObject for specific entities and set attributes using instance variables instead of key paths.
